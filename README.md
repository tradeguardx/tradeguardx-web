# TradeGuardX Landing Page

Modern SaaS landing page for **TradeGuardX** — a browser extension that protects traders from breaking trading rules.

## Tech Stack

- **React** (Vite)
- **TailwindCSS** — dark theme, fintech styling
- **Framer Motion** — section and UI animations

## Requirements

- Node.js **20.x** or **22.x** (required by Vite 7)

## Run locally

```bash
cd tradeguardx-website
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview   # preview production build
```

## Sections

1. **Hero** — Product name, tagline, short description, CTA, illustration
2. **Features** — Rule Enforcement, Auto Trade Protection, Multi-platform, Smart Detection
3. **How it works** — Install → Connect platform → Trade safely
4. **Benefits** — Avoid bans, prop violations, emotional mistakes
5. **Product preview** — UI panel (Symbol, Entry, Current, Stop Loss, P&L)
6. **CTA** — Install extension
7. **Footer** — Features, Documentation, Contact, Privacy

Design: dark theme, Stripe/Vercel/Linear-style fintech SaaS, responsive, subtle animations.
