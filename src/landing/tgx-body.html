Frequently asked

The honest answers.

Will TradeGuardX have access to my funds?

No. We never request withdrawal permissions on your Delta API key Ń only trade and balance read scope. Funds during cooldown are swept to your own Delta sub-account (auto-created by us, controlled by you). When the cooldown ends, the lock lifts and the funds return to your trading wallet.

What happens if the Delta API goes down?

If we lose connection to Delta, you're notified immediately on Telegram and email. Existing positions remain on Delta Ń we don't hold them. When the API is restored, we resume monitoring exactly where we left off. We also run a backup heartbeat that triggers a "full close" if connection is lost for >60 seconds during an active session, configurable per account.

Can I just disable the rules when I want to trade more?

You can change rules in your dashboard, but changes go through a 12-hour cooling-off window Ń you can't loosen a limit while you're in the heat of a bad session. You can tighten any rule instantly. This is intentional. It's the whole point of the product.

Why Delta first? When does CoinDCX go live?

Delta has the best perp API in India for what we need Ń granular, fast, and stable. CoinDCX integration is in active build; we're targeting public launch within 30 days. Pro subscribers get CoinDCX access the day it ships, no upgrade required.

Is this regulated? Is it safe to give you API access?

API keys are scoped (no withdrawal), encrypted at rest, and tied to whitelisted IPs only. We're not a financial advisor and don't custody funds Ń we're a risk-enforcement layer that talks to Delta on your behalf. Read our docs for the full security posture.

Does this work on the Delta mobile app?

Yes. Enforcement is server-side via Delta's API, so it doesn't matter whether you place the trade from web, mobile, or any third-party client. If a trade hits your account, we see it within ~120ms and act if it breaks a rule. Alerts go to your choice of WhatsApp, Telegram, or email Ń you don't need to keep anything open.
